Project Overview
This project analyzes the relationship between trader behavior and Bitcoin market sentiment
(Fear vs Greed) using historical trading data from the Hyperliquid platform.
The objective is to understand how profitability, trading volume, and behavior vary
across different sentiment phases.

Analysis Summary
- Trader data was cleaned and merged with daily market sentiment data.
- Profitability and trading volume were compared across fear and greed phases.
- Visual insights were generated to highlight sentiment-driven behavior patterns.
- All analysis was performed using Google Colab.
